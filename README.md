# Requirements

* opencv-python
* mediapipe
* numpy
* pyautogui
* pycaw
* comtypes

To set up, begin by executing the command `pip install -r requirements.txt` to install the necessary dependencies.

```bash
pip install -r requirements.txt
```

and then run the program using the command `python main.py`.
